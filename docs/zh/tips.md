# 使用技巧

## 4.1基础操作技巧
1. 标高控制：
   - 自由标高：适合复杂地形修改,可以破坏任意位置的方块，同理可以把方块放置在任意高度
   - 锁定标高：创建完美水平面，只能破坏规定标高以上的方块/只能把方块放置在规定标高以下
   - 结合不同高度创建不同要求的地形

2. 显示模式选择：
   - 线框模式：清晰显示操作范围，低性能消耗
   - 表面模式：直观预览效果，适合大范围操作
   - 无显示模式：最优性能，适合熟练用户

3. 撤销/重做使用：
   - 支持最多30步操作历史
   - 使用`Ctrl+Z`快速撤销
   - 使用`Ctrl+Y`重做操作
   - 重启前记得保存重要修改

## 4.2建筑技巧
1. 地形塑造：
   - 先用大范围进行粗略修改
   - 再用小范围精细调整
   - 最后使用平滑模式优化细节

2. 建筑基础：
   - 使用锁定高度模式创建平地
   - 利用长方体模式挖掘地基
   - 结合球形模式创造自然过渡

## 4.3效率提升
1. 快捷键组合：
   - 熟记常用快捷键
   - 建立个人操作习惯
   - 使用配置界面快速切换设置

2. 工作流优化：
   - 合理规划操作顺序
   - 善用撤销/重做系统
   - 适时保存世界备份
